Exemplo de Página HTML - Tags Semânticas e Formulários

Tags Semânticas no HTML

As tags semânticas são importantes para a estrutura e acessibilidade de uma página web. Elas fornecem informações sobre o conteúdo da página, tornando o código mais legível e compreensível.

  • <header> - Usada para definir o cabeçalho da página ou de uma seção.
  • <footer> - Define o rodapé da página ou de uma seção.
  • <article> - Representa um conteúdo independente que pode ser distribuído ou reutilizado.
  • <section> - Define uma seção dentro do conteúdo, geralmente com um título.
  • <nav> - Usada para agrupar links de navegação.
  • <main> - Representa o conteúdo principal da página.

Outras Tags Úteis

Além das tags semânticas e de formulários, o HTML oferece muitas outras tags úteis, como:

  • <img> - Usada para inserir imagens.
  • <a> - Define um link para navegar entre páginas ou recursos.
  • <table> - Usada para criar tabelas.
  • <video> - Para embutir vídeos em uma página.
  • <audio> - Para embutir conteúdo de áudio.

Tabelas

As tags <tr> e <td> são usadas em HTML para criar tabelas.

<tr> – Table Row (linha da tabela)

Representa uma linha dentro de uma tabela.

Fica dentro da tag <table> e geralmente envolve uma ou mais <td> ou <th>(células).

<td> – Table Data (dado da tabela)

Representa uma célula de dado dentro de uma linha.
Deve estar sempre dentro de uma <tr>.

Exemplo de Código

<table>
<tr>
<td>Nome</td>
<td>Idade</td>
</tr>
<tr>
<td>João</td>
<td>25</td>
</tr>
</table>

Listas

As tags <ol> e <ul> são usadas em HTML para criar listas.

<ol> – Ordered List (lista ordenada)

Cria uma lista numerada (com ordem).
Os itens aparecem com números (1, 2, 3...) por padrão.

Exemplo:

<ol>
<li>Primeiro item</li>
<li>Segundo item</li>
<li>Terceiro item</li>
</ol>

Resultado:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

<ul> – Unordered List (lista não ordenada)

Cria uma lista não numerada.

Os itens aparecem com marcadores (bolinhas) por padrão.

Exemplo:

<ul>
<li>Maça</li>
<li>Banana</li>
<li>Laranja</li>
</ul>

Resultado:

  • Maça
  • Banana
  • Laranja
